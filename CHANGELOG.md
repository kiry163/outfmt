## [v0.1.0] - 2026-04-16

### Added

- Initial `outfmt` library with unified `Render` and `Marshal` APIs.
- Support for `table`, `yaml`, and `json` output formats.
- Table rendering for `struct`, `[]struct`, `map[string]any`, and `[]map[string]any`.
- Nested `struct` and `map` flattening for table output with `Parent.Child` columns.
- `outfmt` tags to rename or ignore table columns.
- `WithJSONIndent` and `WithEmptyValue` rendering options.
- Examples for basic usage, CLI integration, and nested data rendering.
- README documentation for installation, API usage, options, and limitations.
